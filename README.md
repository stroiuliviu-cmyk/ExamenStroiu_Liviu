Creați un nou repozitoriu pe GitHub, numit ExamenNume_Prenume. Clonează acest
repozitoriu pe calculatorul local. Adaugă un fișier README.md care să conțină
descrierea proiectului și numărul biletului. Adaugă un fișier .gitignore care să ignore
toate fișierele cu extensia .log. Creează o ramură nouă numită &quot;development&quot;. Adaugă
două fișiere HTML în directorul principal, numite &quot;index.html&quot; și &quot;about.html&quot;. Împingeți
modificările la ramura &quot;development&quot; pe GitHub. Creează o ramură nouă numită
&quot;feature&quot;. Adaugă un fișier CSS numit &quot;style.css&quot; în directorul &quot;css&quot;. Creează un commit
pentru fiecare fișier și director adăugat. Împingeți modificările la ramura &quot;feature&quot; pe
GitHub. Concatenați ramura &quot;feature&quot; cu ramura &quot;development&quot;.
